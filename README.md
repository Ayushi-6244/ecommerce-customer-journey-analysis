# E-Commerce Customer Journey & Conversion Analysis

## Project Overview
This project analyzes customer journey data from an e-commerce platform to understand user behavior, identify checkout drop-offs, and uncover opportunities to improve conversion rates.

Using Python and data analysis techniques, the project evaluates how device usage, referral sources, engagement levels, and cart behavior influence purchasing outcomes.

## Project Notebook
👉 [Open Full Analysis Notebook](notebooks/ecommerce_analysis.ipynb)

## Dataset
The dataset contains over **12,700+ customer interactions**, including:

- Session ID
- Device type
- Referral source
- Country
- Time spent on page
- Items added to cart
- Purchase outcome

## Key Analyses

### Checkout Funnel Analysis
Identified major drop-off points across the customer journey:  
**Home → Product Page → Cart → Checkout → Confirmation**

### Device Performance
Analyzed conversion rates across mobile, desktop, and tablet devices.

### Referral Source Performance
Compared marketing channels such as Google, Email, Social Media, and Direct traffic to identify high-converting sources.

### Engagement Analysis
Examined the relationship between user engagement (time spent) and purchase behavior.

### Cart Behavior Analysis
Analyzed how the number of items added to the cart affects conversion probability.

### Time-Based Analysis
Explored user activity patterns and conversion trends across different hours and days.

## Project Highlights

- Analyzed **12.7K+ customer journey records**
- Identified key drop-off stage between **product page and cart**
- Evaluated behavioral drivers of purchase decisions
- Analyzed conversion trends across **devices, channels, and time**
- Provided actionable recommendations to improve conversion rates

## Tools Used
- Python
- Pandas
- Matplotlib

## Business Insights
The analysis revealed several opportunities to improve conversion rates:

- Improve product page experience to reduce drop-offs before cart stage
- Optimize mobile checkout experience
- Focus marketing investment on high-performing referral channels
- Encourage cart engagement through product recommendations
- Schedule promotions during peak activity hours

## Conclusion
This project demonstrates how data-driven analysis can help e-commerce platforms optimize the customer journey, reduce checkout drop-offs, and improve overall conversion performance.
